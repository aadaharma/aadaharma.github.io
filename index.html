Karelian Embroidery Designer - useful tool for embroidery artists

This is a project I am currently working on: a web application for Karelian embroidery pattern-making. The motivation for this project stemmed from my own embroidery practice. Starting to draft my own patterns, I quickly noticed that there were no ready made tools for Karelian embroidery pattern-making. Drafting on paper is of course quick and easy, but I wanted a way to catalogue common patterns and motifs frequently used in Karelian embroidery.

The application is currently in development and can be used to draft patterns and download them as PDF files. My goal for the future is to add pattern templates and enable the users to make their own templates and share them with the community. You can check out the application here.
